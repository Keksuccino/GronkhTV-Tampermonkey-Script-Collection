# Gronkh.TV Script Collection

Diese kleine Kollection von Tampermonkey Scripts soll das Nutzererlebnis auf Gronkh.TV verbessern, indem es kleine Bugs und Problemchen behebt, und einige QoL Features hinzufügt.

## Chat Filter (`gronkh-tv-chat-filter.user.js`)

Dieses kleine Script fügt eine Filterfunktion für den Chat hinzu, mit der ihr Nachrichten, die bestimmte Wörter oder Wortgruppen enthalten, verstecken könnt. Besonders hilfreich, wenn mal wieder Commands in den Chat gespammt werden.

## Chat Auto-Scroll (`gronkh-autoscroll.user.js`)

In manchen Fällen kann es passieren, dass der Chat nicht ordentlich mitscrollt, vor allem auf größeren (4K) Monitoren. Dieses Script behebt das, indem es einen alternativen Auto-Scroll Modus hinzufügt, der den Chat zum dauerhaften Nach-Unten-Scrollen zwingt.

## Dark Chat Footer (`gronkh-chat-footer-dark.user.js`)

Dieses Script is rein kosmetisch und macht nichts anderes, als dem Chat Footer eine angenehmere, dunkle Farbe zu geben.

<img width="500" alt="Screenshot_7" src="https://github.com/user-attachments/assets/bb022cb6-569b-4cb4-8fd4-fc2f1b6fb6ed" />

## Chat Replay Offset (`gronkh-chat-replay-offset.user.js`)

Bei manchen Streams, vorallem älteren, ist der Chat teilweise sehr asynchron (bis zu 10 Minuten versetzt). Dieses Script behebt das, indem es dich ein Chat-Offset setzen lässt. Das Offset ist pro Stream einstellbar und wird selbst über Browser Neustarts hinweg gespeichert.

<img width="500" alt="Screenshot_8" src="https://github.com/user-attachments/assets/e7c90311-f78a-4816-ad00-78d49a9cdbfd" />

## Twitch Clip Preview (`gronkh-clip-preview.user.js`)

Dieses Script ersetzt Twitch Clip Links im Chat durch echte kleine Vorschau-Cards mit Thumbnail und Clip Titel. Leider noch etwas verbuggt.

## Wide Emote Fix (`gronkh-emote-width-fix.user.js`)

Sehr breite Emotes (meist von 7TV, etc.) werden im Chat Replay nicht richtig angezeigt. Dieses Script behebt das.

## Reload-on-Error (`gronkh-error-reload.user.js`)

Dieses Script lädt automatisch den Stream neu, wenn eine Fehlermeldung (rote Error Nachricht unter Stream) angezeigt wird.

## Nächster/Vorheriger Stream (`gronkh-stream-prev-next.user.js`)

Mit diesem Script werden unter dem Video zwei neue Buttons hinzugefügt, mit denen du problemlos zum vorherigen oder nächsten Stream springen kannst.

<img width="500" alt="Screenshot_11" src="https://github.com/user-attachments/assets/ca220432-a84c-4399-8a51-ad69ed36d5d5" />

## Auto-Play (`gronkh-video-autoplay.user.js`)

Da es seit einem bestimmten Update von Gronkh.TV nicht mehr möglich zu sein scheint, ein Video automatisch starten zu lassen, bringt dieses Script genau dieses alte Auto-Play Verhalten zurück.
